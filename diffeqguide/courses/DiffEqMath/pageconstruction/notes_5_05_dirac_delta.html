Dirac Delta

The Dirac delta function is not a function in a traditional sense. Given some \(a \ge 0,\) it can be interpretted to be \[\delta_a(t) = \left\{\begin{array}{ll} \infty, & \text{if } t = a \\ 0, & \text{otherwise}\end{array}\right.\] Consequently, \[\int_0^\infty g(t)\delta_a(t) \, dt = g(a).\]
