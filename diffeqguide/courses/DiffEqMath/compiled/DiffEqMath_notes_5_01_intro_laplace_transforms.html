Introduction to Laplace Transforms

Introduction

The Laplace transform is an operator that converts differential equations into algebraic equations, which are easier to solve. This transformation also allows us to work with piecewise forcing functions and model a number of real-world situations that were difficult to work with other techniques. The challenge in solving an IVP using the Laplace transform lies in reversing the transformation. In this section we examine the definition and basic properties of the Laplace transform.

Concepts

The Laplace transform of a function \( f(t) \), defined for \(t \geq 0,\) is \[\mathcal{L}\{f(t)\}(s) = F(s) = \int_0^\infty e^{-st} f(t) \, dt.\]
Here are some the Laplace Transform of some important functions.
  • \(\mathcal{L}\{1\} = \dfrac{1}{s}\)

  • \(\mathcal{L}\{t\} = \dfrac{1}{s^2}\)

  • \(\mathcal{L}\{e^{at}\} = \dfrac{1}{s-a}\)

  • \(\mathcal{L}\{\sin(bt)\} = \dfrac{b}{s^2+b^2}\)

  • \(\mathcal{L}\{\cos(bt)\} = \dfrac{s}{s^2+b^2}\)
  • Linearity: \(\mathcal{L}\{af(t) + bg(t)\} = aF(s) + bG(s)\).

  • Derivatives: \(\mathcal{L}\{f'(t)\} = sF(s) - f(0).\)

  • Higher derivatives: \(\mathcal{L}\{f''(t)\} = s^2 F(s) - s f(0) - f'(0).\)
