Module: Laplace Transforms (Introductory)

Learning Goals

  • Understand the definition of the Laplace transform.
  • Compute Laplace transforms of common functions.
  • Apply Laplace transforms to solve initial value problems (IVPs).
  • Use inverse Laplace transforms and basic properties (linearity, shifting).

Concept Review

The Laplace transform of a function \( f(t) \), defined for \( t \geq 0 \), is

\[ \mathcal{L}\{f(t)\}(s) = F(s) = \int_0^\infty e^{-st} f(t) \, dt. \]

Key Properties:

  • Linearity: \(\mathcal{L}\{af(t) + bg(t)\} = aF(s) + bG(s)\).
  • Derivatives: \(\mathcal{L}\{f'(t)\} = sF(s) - f(0).\)
  • Higher derivatives: \(\mathcal{L}\{f''(t)\} = s^2 F(s) - s f(0) - f'(0).\)

Common Transforms:

  • \(\mathcal{L}\{1\} = \dfrac{1}{s}\)
  • \(\mathcal{L}\{t\} = \dfrac{1}{s^2}\)
  • \(\mathcal{L}\{e^{at}\} = \dfrac{1}{s-a}\)
  • \(\mathcal{L}\{\sin(bt)\} = \dfrac{b}{s^2+b^2}\)
  • \(\mathcal{L}\{\cos(bt)\} = \dfrac{s}{s^2+b^2}\)

Practice Problems

A. Core Skill Practice

  1. Compute \(\mathcal{L}\{t^2\}\).
    Hint Recall: \(\mathcal{L}\{t^n\} = \tfrac{n!}{s^{n+1}}\).
    Solution \(\mathcal{L}\{t^2\} = \tfrac{2}{s^3}\).
  2. Compute \(\mathcal{L}\{e^{2t}\cos(3t)\}\).
    Hint Try the shifting rule: \(\mathcal{L}\{e^{at}f(t)\} = F(s-a)\).
    Solution \(\frac{s-2}{(s-2)^2+9}\).
  3. Find the inverse Laplace transform: \(F(s) = \tfrac{3}{s^2+9}\).
    Solution \(\mathcal{L}^{-1}\left(\tfrac{3}{s^2+9}\right) = \sin(3t)\).

B. Solving IVPs with Laplace Transforms

  1. Solve the IVP: \(y''+y=0, \; y(0)=0, \; y'(0)=1.\)
  2. Solve the IVP: \(y'-3y=e^{2t}, \; y(0)=1.\)
  3. Solve the IVP: \(y''+4y'+4y=0, \; y(0)=2, \; y'(0)=0.\)

C. Applications

  1. Mechanical Oscillator: \(y''+4y=0, \; y(0)=0, \; y'(0)=2.\) Use the Laplace transform to find \(y(t)\).
  2. RC Circuit: \(V'(t)+V(t)=E(t)\) with \(E(t)=5\), \(V(0)=0\). Solve for \(V(t)\) given \(R=1\Omega, C=1F\).

D. Challenge Problem

Use the Laplace transform to solve the IVP: \[ y''+y'-2y=\sin(t), \quad y(0)=0, \; y'(0)=1. \] Verify your solution by substitution.

Instructor Notes

  • Core skills reinforce the mechanics of computing Laplace transforms and inverses.
  • IVP problems show the systematic solving process: transform, algebraic manipulation, inverse transform.
  • Application problems demonstrate connections to physics and engineering.
  • The challenge problem highlights how Laplace transforms simplify otherwise tedious ODEs.
